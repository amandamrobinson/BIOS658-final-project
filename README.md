This project consists of the following files:

1. GSE231347_counts.csv.gz: the compressed data file
2. manuscript.Rmd: contains all code for analysis as well as the write up
   input: GSE231347_counts.csv.gz
   output: manuscript.html
2. manuscript.html: the rendered report
3. reference.bib: a list of references in BibTex format
4. vancouver.csl: guide for formatting references
